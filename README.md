# How to run

To run the project run the following command:

```base
python index.py {dataset} {method}
```

The available datasets are:

- `mushrooms`
- `loans`

The available methods are:

- `naive_bayes`
- `decision_tree`
- `random_forest`
- `knn`

Example:

```bash
python index.py mushrooms naive_bayes
```
